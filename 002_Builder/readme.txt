Реализуйте программу для создания различных видов пиццы с использованием шаблона «Строитель». 
В программе должен быть класс «PizzaBuilder», который позволит настраивать параметры пиццы, такие как тесто, соус, сыр, начинка и т. п. 
Метод «buildPizza()» класса «PizzaBuilder» будет возвращать готовую пиццу.
